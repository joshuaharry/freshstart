# Fresh Start

## Motivation

Software engineering in its current form assumes that code and development time
is expensive. Languages like Python, Ruby, and JavaScript made lots of sense
because they made it easy to write programs in a hand-coded environment.
Moreover, mistakes in the lowest levels of the stack were often considered not
worth fixing because of the cost of needing to rebuild our most foundational
abstractions was too high.

Now, in the world of LLMs, many of those assumptions change. So many levels of
the stack are now worth revisiting:

- Programming languages
- Databases
- Text Editors
- Operating systems
- File systems
- Web Browsers

Building new, better things is *easy* now in a way that it never was before;
what happens if you dare to question some of these interfaces and rebuild
them from the ground up?

This project is an attempt to explore what computing can be like when driven
by LLMs. We can build better, more robust software faster now, especially if
we fix the tools to do it; let's see where we can go.
